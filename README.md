# band-store-cth: Full-Stack Application

This repository contains the full source code for the "Chaotic The Harmony" band store e-commerce application, structured as a monorepo.

## 🚀 Key Components

The application is split into two primary components:

1.  **Backend (Java Spring Boot):**

    - Located in the `/backend` folder.
    - Handles all business logic, data persistence, and API endpoints (Maven, Java 21).

2.  **Frontend (React/Vite):**
    - Located in the `/frontend` folder.
    - Provides the user interface and consumes the APIs exposed by the backend (React, npm).
